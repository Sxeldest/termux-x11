# Minimal Code Assistant Agent (prototype)

This folder contains a minimal Python CLI agent scaffold.

Quick start (Windows):

```bat
run_agent.bat scan
```

Quick start (Unix):

```sh
./run_agent.sh scan
```

Commands:
- `scan` — walk the repo and print file-type counts and largest files
- `info` — print agent information

If you'd like, I can now extend the agent to perform automated refactors, open/close PRs, or implement a GitHub Action wrapper. Which capability should I add next?
